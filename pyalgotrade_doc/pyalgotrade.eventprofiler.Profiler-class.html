Class Profiler

source code

This class is responsible for scanning over historical data and analyzing returns before and after the events.

:param predicate: A :class:`Predicate` subclass responsible for identifying events. :type predicate: :class:`Predicate`. :param lookBack: The number of bars before the event to analyze. Must be > 0. :type lookBack: int. :param lookForward: The number of bars after the event to analyze. Must be > 0. :type lookForward: int.
