Class BacktestingStrategy

Base class for backtesting strategies.

:param barFeed: The bar feed to use to backtest the strategy. :type barFeed: :class:`pyalgotrade.barfeed.BaseBarFeed`. :param cash_or_brk: The starting capital or a broker instance. :type cash_or_brk: int/float or :class:`pyalgotrade.broker.Broker`.

.. note:

   This is a base class and should not be used directly.
