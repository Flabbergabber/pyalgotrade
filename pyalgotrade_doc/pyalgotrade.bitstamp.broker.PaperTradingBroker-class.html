Class PaperTradingBroker

source code

A Bitstamp paper trading broker.

:param cash: The initial amount of cash. :type cash: int/float. :param barFeed: The bar feed that will provide the bars. :type barFeed: :class:`pyalgotrade.barfeed.BarFeed` :param fee: The fee percentage for each order. Defaults to 0.5%. :type fee: float.

.. note:

   * Only limit orders are supported.
   * Orders are automatically set as **goodTillCanceled=True** and  **allOrNone=False**.
   * BUY_TO_COVER orders are mapped to BUY orders.
   * SELL_SHORT orders are mapped to SELL orders.
